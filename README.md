# Algos Tests


## Directory Structure

Your directory must be structured as follows:

```
.
├── DD
│   └──Main.java
└── test
    ├── cases
    │   └── DD
    │       └── ...
    ├── __main__.py
    └── README.md
```

Where each directory is named after the problem code.

## Execution

```sh
python3 test <problem_code>
```
